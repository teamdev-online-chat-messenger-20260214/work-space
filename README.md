# Online Chat Messenger
CLIのオンラインチャットツールです。
クライアントはチャットルームの作成または既存のチャットルームへの参加が可能です。チャットルーム作成者とサーバの接続が切れると、そのチャットルームは消失します。

## 使い方
### 1. 前提条件
pythonが必要です。

### 2. インストール
本リポジトリーをクローンします。
```bash
git clone https://github.com/teamdev-online-chat-messenger-20260214/work-space.git
```

### 3. 実行方法
#### 1. サーバ側
以下をターミナルで実行することで、サーバ側のプロセスが立ち上がります。
```bash
python3 server_main.py
```
サーバ側は一度プロセスを立ち上げれば、終了まで操作は必要ありません。

#### 2. クライアント側
以下をターミナルで実行することで、クライアント側のプロセスが立ち上がります。
```bash
python3 client_main.py
```

### 4. 主な操作方法(クライアント側)
#### 1. サーバ側
チャットルームを作成するか、既存のチャットルームに参加するかを選択します。
`1` 入力でチャットルームの作成。 `2`入力で既存のチャットルームへの参加。
```
type 1:create room, 2:join room >
```

#### 2. チャットルーム名の入力
チャットルーム名を入力します。既存のチャットルームに参加する場合は、その名前を入力します。
```
type room name: 
```

#### 3. ユーザ名の入力
ユーザ名を入力します。入力が完了するとそのまま自動でチャットルームへ参加します。
```
type user name: 
```

#### 4. チャット
チャットが開始します。メッセージを入力すると同じ部屋にいるユーザにメッセージが送信されます。
```
>
```
